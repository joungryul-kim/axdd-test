테스트 프로젝트로 AXDD 프레임워크 구성
- 화면을 통한 요구사항 수집 및 관리 

## 개요

이 프로젝트는 AXDD 시스템을 사용하여 개발되었습니다.

## 프로젝트 구조

```
.
├── rules/              # AXDD 프로세스 룰 및 코딩 표준
├── backend/           # 백엔드 애플리케이션
├── frontend/          # 프론트엔드 애플리케이션
└── docs/              # 프로젝트 문서
```

## 시작하기

### 필수 요구사항

- Python 3.11+
- Node.js 18+
- PostgreSQL 14+

- 프레임워크별 개발 가이드
