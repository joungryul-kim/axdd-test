# P03: 결과 제시 프로세스 지침

## 프로세스 개요
- **프로세스 코드**: P03
- **프로세스 이름**: 결과 제시
- **설명**: 분석 결과를 제시하고 검토합니다
- **입력 데이터**: 분석 결과
- **출력 데이터**: 결과 제시 문서

## 프로세스 성격
이 프로세스는 **결과 제시 및 검토** 중심의 프로세스입니다. 분석 결과를 체계적으로 정리하고 제시하는 것이 목적입니다.

## TASK 목록
각 프로세스별로 필요한 TASK를 정의하고, TASK 단위로 작업을 수행합니다.

## TASK별 작업 수행 프롬프트
공통 작업 지침(`00-common-guidelines.md`)과 작업 수행 프롬프트 중요 내용(`04-task-execution-prompt.md`)을 참고하여 작성합니다.

## 프로세스별 특수 지침
- 결과 제시 시 시각화 자료 활용
- 검토 의견 수집 및 반영
- 최종 결과 문서화

## 검증 기준
- [ ] 모든 결과가 제시되었는가?
- [ ] 검토 의견이 수집되었는가?
- [ ] 결과 파일이 올바른 위치에 저장되었는가?
